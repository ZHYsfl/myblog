## 📖 关于本仓库

欢迎来到我的 **数字花园**。

这是一个长期维护的思考存档，记录我在 **AI 基础设施、工程实践、个人成长与创业观察** 等领域的真实思考与一线踩坑经验。

从 LLM 到 Agent，从 Vibe Coding 到 Vibe Research，从单兵作战到团队思维 —— 这里不只是技术笔记，更是一个工程师在快速变化的时代里，通过 **第一性原理**、**源码阅读** 与 **真实世界实践** 去魅求真的完整记录。技术、人生、社交、创业，凡是我认真思考过的，都会写在这里。

### 🌟 核心理念
* **去魅求真**：拒绝盲目跟风，坚持工程现实。读源码 > 看教程，踩坑 > 空想。
* **从单兵到团队**：一个人引擎强不是真的强，能让团队总体引擎强才是真的强。
* **信息即生态**：把社交从资源思维转向生态思维，用真实世界的多元上下文对抗信息茧房。
* **工程纪律**：没有 review 就没有进步，没有 test 就没有质量；用文件体积倒逼解耦，用高测试率锁定逻辑。
