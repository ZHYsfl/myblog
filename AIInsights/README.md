## 📖 About This Repository

Welcome to my **Digital Garden**.

This is a long-term archive of my thinking across **AI infrastructure, engineering practice, personal growth, and entrepreneurship observations**.

From LLMs to Agents, from Vibe Coding to Vibe Research, from solo execution to team thinking — this is not just a collection of technical notes. It is a complete record of an engineer de-mystifying reality through **First Principles**, **Source Code**, and **Real-World Practice** in a rapidly changing era. Technology, life, social connection, and startups: if I've thought seriously about it, it's written here.

### 🌟 Core Philosophy
* **De-mystification**: Reject hype, stick to engineering reality. Read source code > watch tutorials. Learn by failing > learn by imagining.
* **From Solo to Team**: One strong engine is not real strength; making the entire team's engine strong is.
* **Information as Ecosystem**: Shift socializing from a resource mindset to an ecosystem mindset. Use diverse real-world context to fight the filter bubble.
* **Engineering Discipline**: No review, no progress; no test, no quality. Use file size to force decoupling; use high test coverage to lock down logic.
